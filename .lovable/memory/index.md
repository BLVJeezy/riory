# Project Memory

## Core
- **Identity**: RIORY, Belgian infrastructure company. Primary tagline: "Sterk in Rioleringswerk".
- **Tech Stack**: Supabase (DB/Auth/Edge Functions) on Lovable Cloud. WebP images. Dark mode supported.
- **Terminology**: Always use "rijkosten" (not "voorrijkosten"). Use "Afspraak" (not "Gratis Offerte"). Use "Camera inspectie riool".
- **Constraints**: NO AI chatbot. NO WhatsApp link in footer. Do not re-add.
- **Navigation**: Primary CTA always points to `/afspraak`. Auto ScrollToTop on page transitions.
- **SEO**: Strict H1-H2-H3 hierarchy enforced globally. H1 visually shortened, meta title full.
